## Description

Simple CRUD application to manage Teams and Players.
Technologies used: NestJS, TypeORM, ElephantSQL.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run starta

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
